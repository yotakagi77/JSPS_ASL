# Fortran_seminar

1.	テキスト
牛島、“数値計算のためのFortran90/95プログラミング入門”、森北出版

2.	対象
前年度までに同じような演習を行っていないもの(新B4、M1メンター）。

3.	スケジュール
- 4/26(火) 13:00 ガイダンス，演習1.1〜1.24
- 5/10(火) 13:00 演習2.1〜2.24
- 5/17(火) 13:00 演習3.1〜3.34
- 5/24(火) 13:00 演習3.35〜3.41、演習4.1〜4.5、演習5.1〜5.5

- 6/28(火) 13:00 まとめ、報告会

4.	進め方
-	問題一つごとに担当を割り振る(学部3名で1グループ、MC4名で1グループ(樫村はメンター役))
-	当日までにGitHubレポジトリ(Fortran_seminar)に担当したプログラムをアップロードする。
-	ファイル名は、p1_11_takagi.f90（pNo_name.f90）とする。
-	毎週火曜日の勉強会時には課題参加者及び世話役が参加し、世話役がコンパイル・出力を確認し(画面共有)、講評する。基本的に学部生のコードを確認する。
-	最後の6/28報告会では、6章の問題を取りまとめてレポート形式(A4用紙1〜2枚)にして報告する。考察すべきポイントについては世話役が指示する。
