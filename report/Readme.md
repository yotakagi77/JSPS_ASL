# Fortran演習まとめ

## まとめ報告会

## 内容
市坪: 拡散方程式を陽解法・陰解法で解き，各手法の安定性を検証する．格子点数，時間刻みを変更して各計算の計算時間を測定し，各手法の長所・短所を議論する．
菊池: テキストに出てきた反復解法を同じ問題に対して適用し，格子点数を変化させて収束するまでの反復回数・計算時間を整理して優劣を議論する．残差(誤差)と反復回数の履歴グラフも出力して議論する．
住吉: 2次元波動方程式の計算を行い，時間刻み，格子解像度，クーラン数の観点から計算の安定性を示すとともに，計算結果の1例を可視化する．
